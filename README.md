# S3 Synchro Script

Synchronization script for Amazon S3 and compatible solutions (DigitalOcean spaces for example).

## Requirements

* PHP 7.4+

## Setup

```bash
git clone
make install
```

## Tests and commands

See Makefile for all projects or commands:

```bash
make
make test
```
